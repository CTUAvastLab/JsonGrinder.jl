API

JsonGrinder.schema — Function
schema([f=identity,] jsons)

Create schema from an array of jsons optionally mapped by function f.

See also: merge, merge!.

source
JsonGrinder.update! — Function
update!(e, v)

Update the Schema e with value v and return the resulting entry.

source
JsonGrinder.newentry — Function
newentry(v)

Create and return a new Schema according to the type of v and insert v into it.

source
Base.merge — Function
merge(schemas...)

Merge multiple schemas into one.

Useful when for example distributing calculation of schema across multiple workers to aggregate all results.

See also: merge!, schema.

source
Base.merge! — Function
merge!(schema, others...)

Merge multiple schemas others into schema inplace.

See also: merge, schema.
